# Trabajo Integrador – Programación I  
### Manejo de CSV, Diccionarios, Ordenamientos y Estadísticas en Python

---

## Descripción del programa
Este programa carga datos desde un archivo CSV y permite realizar distintas operaciones sobre ellos, tales como:

- Mostrar la lista de países cargados.
- Ordenar los datos mediante diferentes métodos.
- Calcular estadísticas básicas.
- Utilizar diccionarios para organizar información en formato clave:valor.

El objetivo es aplicar los contenidos vistos en Programación I: lectura de archivos, estructuras de datos, funciones, modularización y ordenamientos.

---

## Instrucciones de uso
python src/main.py
El programa mostrará un menú con las siguientes opciones:
1.
2. Mostrar países
3. Ordenar datos
4. Ver Estadísticas básicas
5. Ver diccionario
6. Salir
El usuario debe ingresar el número de la ocpión deseada
